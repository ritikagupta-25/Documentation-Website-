Introduction

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language. Inside a host enviroment (for example, a web browser), JavaScript can be connected objects of its enviroment to provide programmtic control them.

JavaScript contains a standard library of objects, such as Date , and Math , and a core set of language elements such as operators, control structures, and statements. core JavaScript be extended for a variety of purposes by supplementing it additional object; for example:

  • client-side JavaScript extends the core language by supp objects to control a browser and its Document Object Mode For example, client-side extensions allow an application elements on html form and respond to user events such clicks , form input, and page navigation.
